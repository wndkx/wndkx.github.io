CordRPC - is a utility to create a cool menu in your discord profile.

License

Setup

  • Download the CordRPC files
  • Use pip3 install -r requirements.txt to install the requirements

    Editing config file

    Open the config.json file and set all the params

Default config:

{
            "client_id": "YOUR_CLIENT_ID_HERE",
            "large_image": "YOUR_LARGE_IMAGE",
            "large_text": "YOUR_LARGE_TEXT",
            "details": "YOUR_DETAILS",
            "state": "STATE",
            "update_time": 60,
            "buttons": [
                {
                    "label":"BUTTON_LABEL",
                    "url": "URL"
                },
                {
                    "label":"BUTTON_LABEL",
                    "url": "URL"
                }
            ]
        }
  • client_id - You can find the client id in Discord Developer Portal/Your application/Oauth2/General
  • large_image - This image will be displayed at Activity menu in your profile. You can edit it at Discord Developer Portal/Your application/Rich Presence/ Art assets. NOTE! On some clients the image won't seem
  • large_text - This parameter is a text that will be displayed when you point at the image
  • update_time - Time after menu will disappear
  • Other params you can see here
